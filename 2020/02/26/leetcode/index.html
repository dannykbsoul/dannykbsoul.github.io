  2. "132"
  3. "213"
  4. "231"
  5. "312"
  6. "321"

Given n and k, return the kth permutation sequence.

  • Given n will be between 1 and 9 inclusive.

思路

如果直接把所有的排列组合算出来再输出第k个组合的话，时间和空间复杂度会很高。

我们可以观察到：

n=3时

  • 123

  • 213

发现开头的数字俩俩一组，即(n-1)!所以我们可以对k进行操作，k/(n-1)!是第几组，即确定了前面的数字，k%(n-1)!是下个递归的开始。
